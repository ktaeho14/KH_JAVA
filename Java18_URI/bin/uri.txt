1. URL : Uniform Resource Locator  주소(위치)
URL은 특정 서버의 한 리소스에 대한 구체적인 위치를 서술

https://www.lge.co.kr/lgekor/product/pc/notebook/productList.do?cateId=2170

2. URN : Uniform Resource Name 대상(이름)
URN은 콘텐츠를 이루는 리소스에 대해, 그 리소스의 위치에 영향 받지 않는 유일무이한 이름 역할


3. URI : Uniform Resource Identifier 식별자
URI = URL + URN
